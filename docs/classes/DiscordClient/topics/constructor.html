Classes DiscordClient Initializer
Initializer

DiscordClient(options: {...args})

Returns a DiscordClient instance.

Declaration

new DiscordClient(options: {...args})

Parameters

options

Options supplied to the client. This should be an object of arguments

{...args}

Possible arguments;

  • token: string - A Discord Bot Token
Important
A token attribute must be supplied as part of the options and must have a value with type string, corresponding with a valid Discord API Bot Token

Return Value

A new DiscordClient instance.

Discussion

This instance servers as the entry point for this module and most interactions will occur through it. Whilst it is possible to run multiple instances at once with separate tokens within the same code, it is discouraged due to potentially running into resource (such as memory) issues. This is, however, down to the developer to decide and depends on the scope of the intended project.
