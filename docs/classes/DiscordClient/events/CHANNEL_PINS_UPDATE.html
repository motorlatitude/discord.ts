Event

CHANNEL_PINS_UPDATE

Event is emitted if a message is pinned or unpinned.

Declaration

event: "CHANNEL_PINS_UPDATE"

Parameters

ChannelPinsObject: IChannelPinsUpdateEventObject

An object containing the relevant Guild and Channel instances as well as the LastPinTimestamp value.

Discussion

This event is emitted if a message has been pinned or unpinned in a channel that bot is a member off. If this channel is part of a guild the Channel parameter will be of type TextChannel, otherwise if the channel is a direct message channel the Channel parameter will be of type DirectMessageChannel.
