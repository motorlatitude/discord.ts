Classes DiscordClient Disconnect
Instance Method

function Disconnect()

Close the active gateway connection.

Declaration

function Disconnect(): void

Discussion

This method will go and close the active gateway connection. User-specified variables will still contain information, this may however be outdated after disconnect.

This method will trigger the ClientConnection instance to disconnect and the DiscordClient instance will emit a DISCONNECT event once the connection has successfully been closed.
