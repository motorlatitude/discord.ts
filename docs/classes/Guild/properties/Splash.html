Classes Guild Splash
Instance Property

Splash

The splash hash of the guild.

Declaration

var Splash?: string

Discussion

This property contains the splash hash of the guild splash and can be found at discord cdn;

Discord's Splash CDN URL
https://cdn.discordapp.com/splashes/{guild_id}/{splash_hash}.png

The property is undefined if the guild does not have a splash set.
