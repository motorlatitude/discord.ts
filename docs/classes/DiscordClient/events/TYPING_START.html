Classes DiscordClient Events TYPING_START
Event

TYPING_START

Event is emitted when a user starts typing in a channel.

Declaration

event: "TYPING_START"

Parameters

Channel: TextChannel | DirectMessageChannel

The relevant channel instance for where the user has started typing

User: User

A User instance describing the user who has started typing

Timestamp: number

UNIX timestamp (in seconds) of when the user started typing

Guild?: Guild

A Guild instance where the channel resides

Discussion

This event is emitted when a user starts typing in a channel in which the bot user is a member of. This could be a TextChannel or DirectMessage instance.
