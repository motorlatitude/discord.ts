Classes DiscordClient Events GUILD_MEMBER_REMOVE
Event

GUILD_MEMBER_REMOVE

Event is emitted if someone leaves a guild.

Declaration

event: "GUILD_MEMBER_REMOVE"

Parameters

Guild: Guild

A Guild instance containing the existing updated Guild instance.

GuildMember: GuildMember

The GuildMember instance of the left member

Discussion

This event is emitted if a user leaves a guild that the bot is a member of. This leave could be of their own will, being kicked or banned.
