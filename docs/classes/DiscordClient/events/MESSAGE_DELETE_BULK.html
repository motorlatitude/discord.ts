Classes DiscordClient Events MESSAGE_DELETE_BULK
Event

MESSAGE_DELETE_BULK

Event is emitted when multiple messages are deleted in a channel.

Declaration

event: "MESSAGE_DELETE_BULK"

Parameters

Messages: Message[]

An array containing the old Message instances.

Channel: TextChannel | DirectMessageChannel

The channel instance describing the channel in which the messages were created

Guild?: Guild

The Guild instance in which the TextChannel instance resides

Discussion

This event is emitted when multiple message are delete in a channel that the bot user is a member of.

The Guild parameter will only be populated if the message was sent within a guild. Otherwise the message originates as a direct message, and the Channel parameter will hold a DirectMessageChannel instances and the Guild parameter will be undefined.
