Classes Guild ExplicitContentFilter
Instance Property

ExplicitContentFilter

Defines which users' messages are scanned for explicit content.

Declaration

var ExplicitContentFilter: number

Discussion

This property corresponds to which messages from which users to scan for explicit content. The value is a number between and including 0 - 2.

Level Integer Description
DISABLED 0 No messages are scanned for explicit content
MEMBERS_WITHOUT_ROLES 1 Only messages sent by users without a role are scanned for explicit content
ALL_MEMBERS 2 Scan all messages for explicit content
