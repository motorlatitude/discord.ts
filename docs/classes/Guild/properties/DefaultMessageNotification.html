Classes Guild DefaultMessageNotification
Instance Property

DefaultMessageNotification

The default message notification threshold.

Declaration

var DefaultMessageNotification: number

Discussion

The default notification level for this guild. A value that is either 0 or 1.

Level Integer Description
ALL_MESSAGES 0 Be notified about all messages
ONLY_MENTIONS 1 Only be notified about messages where the user is specifically mentioned
