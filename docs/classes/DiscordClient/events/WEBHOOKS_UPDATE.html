Event

WEBHOOKS_UPDATE

Event is emitted when a webhook in a channel is modified.

Declaration

event: "WEBHOOKS_UPDATE"

Parameters

Channel: TextChannel

The TextChannel instance where the webhook has been modified

Guild: Guild

The Guild instance in which the TextChannel resides

Discussion

This event is emitted when a webhook is created, updated or deleted in a guild's text channel.
