Event

VOICE_STATE_UPDATE

Event is emitted when someones voice state updates.

Declaration

event: "VOICE_STATE_UPDATE"

Parameters

EventType: "JOINED" | "UPDATED" | "LEFT"

The event type

VoiceState: VoiceState

The new VoiceState instance.

Discussion

This event is emitted when a user joins / leaves / moves voice channels. The EventType will tell you if a user has joined or left a channel. The "UPDATED" EventType is set if the user is already in a voice channel and has either moved, set them self to deaf, set them self to mute, been server deafened, been server muted or whether the current bot user has muted (suppressed) them.
