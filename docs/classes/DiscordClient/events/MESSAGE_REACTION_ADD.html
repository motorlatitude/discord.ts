Event

MESSAGE_REACTION_ADD

Event is emitted when a reaction is added to a message.

Declaration

event: "MESSAGE_REACTION_ADD"

Parameters

Channel: TextChannel | DirectMessageChannel

The channel instance describing the channel in which the message was created

MessageId: string

The message id of the relevant message.

Emoji: ReactionEmoji

The emoji used to react to the message.

Guild?: Guild

The Guild instance in which the TextChannel instance resides

Discussion

This event is emitted when someone reacts to a message in a channel that the bot user is a member of.

The Guild parameter will only be populated if the message was sent within a guild. Otherwise the message originates as a direct message, and the Channel parameter will hold a DirectMessageChannel instances and the Guild parameter will be undefined.

Information
The returned emoji is not a standard Emoji instance. It is instead a ReactionEmoji instance and will contain only the required Name and optional id properties.

The full message instance should be requested from the Channel parameter.
