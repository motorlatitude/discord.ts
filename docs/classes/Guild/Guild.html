class Guild

Overview

The Guild instance corresponds to a single Discord guild. It will contain all relevant properties of a Discord guild and allows API methods to be called from it through the use of Actions.

Actions

Actions are methods that can be called from the Guild instance and will call Discord's REST/HTTP API. The Actions will always return a promise resolve or reject based on if the request was successful or unsuccessful. The values of these requests will not be cached, however, a large number of these requests will trigger gateway events to fire, these will update the library's state.
