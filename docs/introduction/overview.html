Introduction Overview

DiscordTS Documentation

Welcome to the DiscordTS documentation. DiscordTS is a NPM module written in typescript to communicate with the Discord API. These pages are dedicated towards utilising this module to its full potential and how to go about creating a fully functional discord bot.

This is the documentation for DiscordTS version v0.1.0.

Getting Started

If you're new, you might find these links helpful;

  • You can familiarise yourself with discord api over at: Discord API Documentation
  • Check out our Getting Started section for help using DiscordTS
  • You can ask for help or just for a chat in our Discord Server

Reading These Docs

This documentation is laid out to make it as easy as possible to find what you're looking for. These docs have split the various components of DiscordTS into separate classes and can be read individually by using the sidebar on the left. The main class and entry point for this module is the DiscordClient class. Each class will provide a declaration and an overview of its uses and will furthermore be subdivided into individual topics that will cover all available parameters, methods or events of a class.
