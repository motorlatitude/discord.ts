Classes DiscordClient Events MESSAGE_DELETE
Event

MESSAGE_DELETE

Event is emitted when a message is deleted in a channel.

Declaration

event: "MESSAGE_DELETE"

Parameters

Message: Message

A Message instance for the old message

Channel: TextChannel | DirectMessageChannel

The channel instance describing the channel in which the message was created

Author: User

A User instance for the person sending the message

Guild?: Guild

The Guild instance in which the TextChannel instance resides

GuildMember?: GuildMember

The GuildMember instance for the author

Discussion

This event is emitted when a message has been deleted in a channel that the bot user is a member of.

The Guild and GUILD_MEMBER parameters will only be populated if the message was sent within a guild. Otherwise the message originates as a direct message, and the Channel parameter will hold a DirectMessageChannel instances and the Guild and GuildMember parameters will both be undefined.
