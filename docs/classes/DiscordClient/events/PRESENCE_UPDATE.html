Classes DiscordClient Events PRESENCE_UPDATE
Event

PRESENCE_UPDATE

Event is emitted when a user's presence or information changes.

Declaration

event: "PRESENCE_UPDATE"

Parameters

NewPresence: Presence

The updated Presence instance

OldPresence?: Presence

The previous presence object, this is not present if no previous presence exists for the user

Discussion

This event is emitted when a user's presence or information such as name or avatar are updated.

Careful
This event emitted from discord could only hold partial information and the Presence instance may not have all properties as a result, take care and do not assume that all properties have values.
