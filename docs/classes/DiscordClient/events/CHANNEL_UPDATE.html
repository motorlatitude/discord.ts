Classes DiscordClient Events CHANNEL_UPDATE
Event

CHANNEL_UPDATE

Event is emitted if a new channel has been updated.

Declaration

event: "CHANNEL_UPDATE"

Parameters

Channel: TextChannel | VoiceChannel | CategoryChannel | DirectMessageChannel

A Channel instance containing the existing updated channel instance.

Discussion

This event is emitted if a channel is updated in a guild in which the bot is a member and will return an existing TextChannel, VoiceChannel or CategoryChannel instance. It will also be emitted if a DM channel is updated and will return an existing DirectMessageChannel instance in this case.
