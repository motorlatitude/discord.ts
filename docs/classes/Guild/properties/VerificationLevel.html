Classes Guild VerificationLevel
Instance Property

VerificationLevel

The verification level required to send messages.

Declaration

var VerificationLevel: number

Discussion

The minimum level a user must be verified for them to be allowed to send messages in a text channel or to initiate direct message conversations. If the user is assigned a role this will not apply. It corresponds to a number value between and including 0 - 4.

Level Integer Description
NONE 0 Unrestricted, any user can join and send messages
LOW 1 The user must have a verified email associated with their account
MEDIUM 2 A user must have been registered on discord for longer than 5 minutes
HIGH 3 A user is required to have been a member of the server for more than 10 minutes
VERY_HIGH 4 A user is required to also have a verified phone number

The higher levels will include the requirements of the lower levels as well.
