Classes DiscordClient Events VOICE_SERVER_UPDATE
Event

VOICE_SERVER_UPDATE

Event is emitted when joining or switching voice channels.

Declaration

event: "VOICE_SERVER_UPDATE"

Parameters

VoiceHandler: VoiceHandler

The new VoiceHandler tied to that voice channel

Discussion

This event is emitted when a guild's voice server is updated. It is sent when initially connecting to a voice channel and also when the current voice instance fails over to a new server.
