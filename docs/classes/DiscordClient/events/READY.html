Classes DiscordClient Events READY
Event

READY

Event is emitted once the client is ready.

Declaration

event: "READY"

Parameters

User: User

A User instance for the current connections user

Discussion

This event is emitted once a suitable gateway WebSocket URL was fetched and a new WebSocket connection was established and the connection was opened. An identify packet was already sent to discord and accepted with a READY event emitted from the active Discord gateway WebSocket connection.

This event should be used to establish that the client is ready to handle WebSocket communications and no communications should be sent until then. Should the client disconnect a DISCONNECT event will be emitted.

This user instance can now be accessed from the client as long as the client is connected; 

const DiscordClient = require('discordts');

DiscordClient.on("READY", function(UserInstance: User){
    console.log(UserInstance.id);
    // Will output the user id
    console.log(DiscordClient.User.id);
    // Will output the same user id
});
