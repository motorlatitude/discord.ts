Classes Guild MFALevel
Instance Property

MFALevel

The MFA level for a guild

Declaration

var MFALevel: number

Discussion

The required MFA level for a guild, i.e. is a user required to have set up multi-factor authentication in this guild? The value can either be 0 or 1.

Level Integer Description
NONE 0 Users are NOT required to have set up multi-factor authentication
ELEVATED 1 Users ARE required to have set up multi-factor authentication
