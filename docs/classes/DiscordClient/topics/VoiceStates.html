Classes DiscordClient VoiceStates
Instance Property

VoiceStates

The VoiceStates for non-guild voice connections.

Declaration

var VoiceStates: VoiceStateStore

Discussion

This property stores the VoiceState instance's that are not part of a guild in a VoiceStateStore instance.

Note
This property will contain no VoiceStates until the READY event has been sent.
