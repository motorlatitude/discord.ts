Classes Guild PremiumTier
Instance Property

PremiumTier

Which boost level the guild has

Declaration

var PremiumTier: number

Discussion

Which boost tier the guild has reached. The value can be a number between and including 0 - 3.

Level Integer Description
NONE 0 Has not reached a tier
TIER_1 1 Has reached the first tier of server boosting
TIER_2 2 Has reached the second tier of server boosting
TIER_3 3 Has reached the third and final tier of server boosting
