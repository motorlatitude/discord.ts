DiscordClient

An instance that controls events and initiates a connection with the discord gateway server.

Declaration

class DiscordClient extends events.EventEmitter

Overview

Every application wanting to utilise discords API gateway will require to use a DiscordClient instance to keep track of Channels, Guilds, Users and more. The DiscordClient instances will issue events as they come in from the gateway after manipulating its internal components.

The DiscordClient instance can also be used to access discords REST API irrespective of using the gateway connection.
