Classes DiscordClient Events GUILD_DELETE
Event

GUILD_DELETE

Event is emitted if a guild becomes unavailable.

Declaration

event: "GUILD_DELETE"

Parameters

DeletedGuild: IGuildDeleteEventObject

Information about the removal.

IGuildDeleteEventObject

An object containing id, Unavailable? and WasRemoved

Discussion

This event is emitted if a guild becomes unavailable due to a guild outage, or when the bot leaves or is removed from the guild.

WasRemoved argument will tell you if the bot was kicked or if there's a guild outage.
