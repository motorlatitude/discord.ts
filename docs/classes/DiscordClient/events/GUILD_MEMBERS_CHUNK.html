Event

GUILD_MEMBERS_CHUNK

Event returns a list of guild members.

Declaration

event: "GUILD_MEMBERS_CHUNK"

Parameters

Guild: Guild

A Guild instance containing the existing updated Guild instance.

GuildMembers: GuildMember[]

An array of GuildMember instances.

Discussion

This event is emitted in response to requesting guild members via the gateway. This will automatically also completely populate the Members attribute of the Guild instance.
