Classes DiscordClient Events CHANNEL_DELETE
Event

CHANNEL_DELETE

Event is emitted if a channel is removed.

Declaration

event: "CHANNEL_DELETE"

Parameters

ChannelDeleteObject: IChannelDeleteEventObject

A Channel instance containing the existing updated channel instance.

Discussion

This event is emitted if a channel is deleted in a guild in which the bot is a member. It will also be emitted if a DM channel is removed.
