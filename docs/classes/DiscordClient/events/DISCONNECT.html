Classes DiscordClient Events DISCONNECT
Event

DISCONNECT

Event is emitted once the connection has closed.

Declaration

event: "DISCONNECT"

Parameters

This event does not return any values

Discussion

This event is emitted after the connection to the Gateway has been closed. This occurs when the connection to the gateway server has been terminated.

Note
The DISCONNECT event is NOT final. Further events may trigger as part of the library attempting to reconnect, unless the disconnect was triggered through the .Disconnect() instance method.
